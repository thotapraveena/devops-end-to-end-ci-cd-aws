Project Overview:-
This project demonstrates a complete DevOps workflow including Infrastructure as Code, containerization, CI/CD automation, and AWS deployment.

Technologies Used:-
AWS (EC2, VPC, S3, IAM, ECR, ECS)
Terraform
Docker & Docker Compose
Jenkins
Git
Linux

Architecture Flow:-
Infrastructure provisioned using Terraform
Application containerized using Docker
Jenkins pipeline automates build & deployment
Docker image pushed to AWS ECR
Application deployed on EC2 / ECS
NGINX configured as reverse proxy

Key Features:-
Automated infrastructure provisioning
CI/CD pipeline automation
Containerized deployment
Modular Terraform configuration

Outcome:-
Reduced manual deployment effort by automating infrastructure provisioning and application deployment workflow.
